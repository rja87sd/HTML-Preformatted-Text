Exploring Pre-Formatted Text

What is the <pre> element?

The <pre> element stands for "Pre-Formatted Text".


Why is it used?

The <pre> element is used when the developer wishes to include spacing and line breaks in text without having to use CSS styling or JavaScript to format the text themselves.

Example of non-formatted text:

for (const key in data) { let para = document.createElement('p'); para.innerText = `${key}: ${data[key]}`; profileContainer.appendChild(para); }


Example of pre-formatted text:

                    for (const key in data) {
                    let para = document.createElement('p');
                    para.innerText = `${key}: ${data[key]}`;
                    profileContainer.appendChild(para);
                    }
